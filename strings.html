PHP Strings

PHP Strings

A string is a sequence of characters, like "Hello World!". Strings in PHP can be created using single quotes, double quotes, or heredoc syntax.

1. Creating Strings

There are several ways to create strings in PHP:

<?php
// Single quotes
$str1 = 'Hello World!';

// Double quotes
$str2 = "Hello World!";

// Heredoc syntax
$str3 = <<<EOT
This is a heredoc string
It can span multiple lines
EOT;

echo $str1 . "<br>";
echo $str2 . "<br>";
echo $str3;
?>

2. String Length

Use the strlen() function to get the length of a string: 

<?php
$text = "Hello World!";
echo "String length: " . strlen($text); // Output: 12
?>

3. String Functions

PHP provides many built-in functions for string manipulation:

3.1 strtolower() and strtoupper()

<?php
$text = "Hello World!";
echo strtolower($text); // Output: hello world!
echo "<br>";
echo strtoupper($text); // Output: HELLO WORLD!
?>

3.2 ucfirst() and ucwords()

<?php
$text = "hello world!";
echo ucfirst($text); // Output: Hello world!
echo "<br>";
echo ucwords($text); // Output: Hello World!
?>

3.3 str_replace()

<?php
$text = "Hello World!";
$new_text = str_replace("World", "PHP", $text);
echo $new_text; // Output: Hello PHP!
?>

3.4 substr()

<?php
$text = "Hello World!";
echo substr($text, 0, 5); // Output: Hello
echo "<br>";
echo substr($text, 6); // Output: World!
?>

4. String Concatenation

You can concatenate strings using the dot (.) operator:

<?php
$first = "Hello";
$second = "World";
$result = $first . " " . $second;
echo $result; // Output: Hello World

// Using .= operator
$text = "Hello";
$text .= " World!";
echo $text; // Output: Hello World!
?>

5. String Comparison

You can compare strings using various operators and functions:

<?php
$str1 = "Hello";
$str2 = "Hello";
$str3 = "hello";

// Using == operator
if ($str1 == $str2) {
    echo "Strings are equal<br>";
}

// Using strcmp() function
if (strcmp($str1, $str2) == 0) {
    echo "Strings are identical<br>";
}

// Case-insensitive comparison
if (strcasecmp($str1, $str3) == 0) {
    echo "Strings are equal (case-insensitive)<br>";
}
?>

6. String Searching

Find substrings within strings using various functions:

<?php
$text = "Hello World!";

// strpos() - find position of substring
$pos = strpos($text, "World");
echo "Position of 'World': " . $pos; // Output: 6

// stripos() - case-insensitive search
$pos = stripos($text, "world");
echo "<br>Position of 'world' (case-insensitive): " . $pos; // Output: 6

// str_contains() - check if string contains substring
if (str_contains($text, "World")) {
    echo "<br>String contains 'World'";
}
?>

7. String Formatting

Format strings using sprintf() and printf():

<?php
$name = "John";
$age = 25;

// Using sprintf()
$message = sprintf("My name is %s and I am %d years old.", $name, $age);
echo $message;

// Using printf()
printf("<br>My name is %s and I am %d years old.", $name, $age);
?>

8. String Trimming

Remove whitespace from strings:

<?php
$text = "  Hello World!  ";

echo "Original: '" . $text . "'<br>";
echo "Trimmed: '" . trim($text) . "'<br>";
echo "Left trimmed: '" . ltrim($text) . "'<br>";
echo "Right trimmed: '" . rtrim($text) . "'<br>";
?>

9. String Splitting and Joining

Split strings into arrays and join arrays into strings:

<?php
$text = "apple,banana,orange";

// Split string into array
$fruits = explode(",", $text);
print_r($fruits);

// Join array into string
$new_text = implode(" and ", $fruits);
echo "<br>" . $new_text; // Output: apple and banana and orange
?>

10. HTML Special Characters

Convert special characters to HTML entities:

<?php
$text = "<script>alert('Hello')</script>";

// Convert to HTML entities
$safe_text = htmlspecialchars($text);
echo $safe_text; // Output: &lt;script&gt;alert('Hello')&lt;/script&gt;

// Convert back
$original = htmlspecialchars_decode($safe_text);
echo "<br>" . $original;
?>
