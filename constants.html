PHP Constants

PHP Constants

In PHP, constants are like variables, but once they are defined, their values cannot be changed or undefined. Constants are useful for storing values that should remain the same throughout the execution of a script, such as configuration values, API keys, or mathematical constants.

How to define a constant in PHP?

  • Using define() function: 
    <?php
define("SITE_NAME", "MyWebsite");
echo SITE_NAME; // Output: MyWebsite
?>

    Note: define() is used outside of classes and can be used anywhere in your code, even inside functions.

  • Using const keyword: 
    <?php
const PI = 3.1416;
echo PI; // Output: 3.1416
?>

    Note: const is a language construct and is often used to define constants inside classes or at the top level of a script (not inside functions or loops).

Key Points about Constants

  • Constant names are usually written in uppercase letters by convention.
  • Constants do not start with a $ sign.
  • Constants are automatically global and can be used across the entire script.
  • Once set, the value of a constant cannot be changed or undefined.

Example: Using Constants in PHP

<?php
define("GREETING", "Welcome to PHP!");
const VERSION = "8.2";

echo GREETING; // Output: Welcome to PHP!
echo VERSION;  // Output: 8.2
?>

বাংলা ব্যাখ্যা (Bangla Explanation)

PHP-তে constant হচ্ছে এমন একটি মান, যেটি একবার নির্ধারণ করার পর পরিবর্তন করা যায় না। সাধারণত define() ফাংশন অথবা const কীওয়ার্ড ব্যবহার করে constant তৈরি করা হয়।
উদাহরণ: 

<?php
define("APP_NAME", "আমার অ্যাপ");
const COUNTRY = "বাংলাদেশ";

echo APP_NAME;   // আউটপুট: আমার অ্যাপ
echo COUNTRY;    // আউটপুট: বাংলাদেশ
?>
  • constant-এর নাম সাধারণত বড় হাতের অক্ষরে লেখা হয়।
  • constant-এর আগে $ চিহ্ন ব্যবহার হয় না।
  • একবার মান সেট করলে তা পরিবর্তন করা যায় না।
