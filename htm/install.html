PHP Installation Guide

Set up your PHP development environment

How to Install PHP

PHP can be installed in several ways depending on your operating system and requirements. The easiest way for beginners is to install a complete web server package that includes PHP, MySQL, and Apache.

üéØ Recommended: For beginners, we recommend installing XAMPP, WAMP, or MAMP as they provide a complete development environment with PHP, MySQL, and Apache server.

Method 1: XAMPP (Cross-Platform)

XAMPP is the most popular and easiest way to install PHP. It works on Windows, macOS, and Linux.

Installation Steps:

  1. Download XAMPP: Visit apachefriends.org
  2. Choose Version: Download the version for your operating system
  3. Run Installer: Execute the downloaded file and follow the installation wizard
  4. Start Services: Open XAMPP Control Panel and start Apache and MySQL
  5. Test Installation: Open browser and go to http://localhost

Method 2: WAMP (Windows)

WAMP is specifically designed for Windows systems.

Installation Steps:

  1. Download WAMP: Visit wampserver.com
  2. Install WAMP: Run the installer and follow the setup process
  3. Start WAMP: Launch WAMP from the system tray
  4. Access Localhost: Go to http://localhost in your browser

Method 3: MAMP (macOS)

MAMP is designed for macOS users.

Installation Steps:

  1. Download MAMP: Visit mamp.info
  2. Install MAMP: Drag MAMP to Applications folder
  3. Start MAMP: Open MAMP and click "Start Servers"
  4. Access Web Server: Click "Open WebStart page" or go to http://localhost:8888

Method 4: Manual Installation

For advanced users who want to install PHP manually:

Windows Manual Installation:

# Download PHP from php.net
# Extract to C:\php
# Add C:\php to PATH environment variable
# Configure php.ini file
# Install Apache or Nginx web server

Linux (Ubuntu/Debian):

sudo apt update
sudo apt install php php-mysql apache2
sudo systemctl start apache2
sudo systemctl enable apache2

macOS (using Homebrew):

brew install php
brew services start php

Testing Your PHP Installation

After installation, create a test file to verify PHP is working correctly: 

<?php
// Create a file named test.php in your web server directory
echo "Hello, PHP is working!";
echo "<br>";
echo "PHP Version: " . phpversion();
echo "<br>";
echo "Current Date: " . date("Y-m-d H:i:s");
?>

Directory Structure

After installation, your web files should be placed in the appropriate directory:

  • XAMPP: C:\xampp\htdocs\ (Windows) or /Applications/XAMPP/htdocs/ (macOS)
  • WAMP: C:\wamp64\www\ or C:\wamp\www\
  • MAMP: /Applications/MAMP/htdocs/
  • Linux Apache: /var/www/html/
