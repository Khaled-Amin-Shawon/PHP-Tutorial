PHP Switch

PHP Switch

The switch statement in PHP is used to perform different actions based on different conditions. It is an alternative to using multiple if...elseif...else statements when you need to compare the same variable or expression with many different values.

Syntax

<?php
switch (expression) {
    case value1:
        // code to execute if expression == value1
        break;
    case value2:
        // code to execute if expression == value2
        break;
    // more cases...
    default:
        // code to execute if no case matches
}
?>

Example

<?php
$day = "Tuesday";

switch ($day) {
    case "Monday":
        echo "Today is Monday";
        break;
    case "Tuesday":
        echo "Today is Tuesday";
        break;
    case "Wednesday":
        echo "Today is Wednesday";
        break;
    default:
        echo "It's another day";
}
?>
  • case: Each case checks a value. If it matches, the code block runs and stops at the break statement.
  • default: If no case matches, the default block runs.

When to Use switch?

  • When you have a single variable/expression to compare against multiple possible values.
  • It makes code cleaner and easier to read than many if...elseif...else statements.
