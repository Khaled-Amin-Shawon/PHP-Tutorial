PHP Functions

PHP Functions

In PHP, a function is a block of code that performs a specific task and can be reused throughout your script. You define a function using the function keyword, followed by the function name and parentheses. You can also pass information to functions using parameters.

How to Define and Call a Function

<?php
function sayHello() {
    echo "Hello from a function!";
}
sayHello(); // Output: Hello from a function!
?>
Hello from a function!

You can also pass parameters to a function: 

<?php
function greet($name) {
    return "Hello, $name!";
}
echo greet("PHP"); // Output: Hello, PHP!
?>
Hello, PHP!

Bangla Explanation

PHP-তে function হলো কোডের এমন একটি অংশ, যেটি নির্দিষ্ট কাজ করে এবং বারবার ব্যবহার করা যায়। function কীওয়ার্ড দিয়ে ফাংশন ডিফাইন করতে হয়। ফাংশনে প্যারামিটারও পাঠানো যায়। 

<?php
function sayHello() {
    echo "একটি ফাংশন থেকে হ্যালো!";
}
sayHello(); // আউটপুট: একটি ফাংশন থেকে হ্যালো!
?>
একটি ফাংশন থেকে হ্যালো!

প্যারামিটারসহ ফাংশন: 

<?php
function greet($name) {
    return "হ্যালো, $name!";
}
echo greet("পিএইচপি"); // আউটপুট: হ্যালো, পিএইচপি!
?>
হ্যালো, পিএইচপি!
