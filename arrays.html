PHP Arrays

Arrays are used to store multiple values in a single variable. There are indexed arrays and associative arrays.

Indexed Array

<?php
    $fruits = ["Apple", "Banana", "Cherry"];
    echo $fruits[0]; // Apple
?>
Apple

Associative Array

<?php
    $ages = ["Peter" => 22, "John" => 25];
    echo $ages["John"]; // 25
?>
25
