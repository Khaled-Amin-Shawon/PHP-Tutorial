PHP Numbers

PHP supports integers and floats (decimals). You can use arithmetic operators to perform calculations.

<?php
    $a = 10;      // Integer
    $b = 4.5;     // Float
    $sum = $a + $b;
    $mul = $a * $b;
    echo $sum;    // 14.5
    echo $mul;    // 45
?>
