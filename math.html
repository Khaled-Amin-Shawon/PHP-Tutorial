PHP Math

PHP provides a wide range of built-in mathematical functions to perform calculations, such as addition, subtraction, rounding, generating random numbers, and more. These functions make it easy to handle mathematical operations in your PHP scripts.

Common PHP Math Functions

  • abs(x) — Returns the absolute (positive) value of x
  • pow(x, y) — Returns x raised to the power of y
  • sqrt(x) — Returns the square root of x
  • round(x) — Rounds x to the nearest integer
  • ceil(x) — Rounds x up to the nearest integer
  • floor(x) — Rounds x down to the nearest integer
  • min(x, y, ...) — Returns the lowest value
  • max(x, y, ...) — Returns the highest value
  • rand(min, max) — Returns a random integer between min and max
  • pi() — Returns the value of π (pi)

Example: Basic Math Functions in PHP

<?php
echo abs(-15);         // Output: 15
echo pow(2, 3);        // Output: 8
echo sqrt(16);         // Output: 4
echo round(3.7);       // Output: 4
echo ceil(3.2);        // Output: 4
echo floor(3.8);       // Output: 3
echo min(2, 5, 1, 9);  // Output: 1
echo max(2, 5, 1, 9);  // Output: 9
echo rand(1, 100);     // Output: (random number between 1 and 100)
echo pi();             // Output: 3.1415926535898
?>
