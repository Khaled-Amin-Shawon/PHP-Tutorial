PHP Math

PHP provides many built-in math functions, such as abs(), pow(), sqrt(), round(), etc. 

<?php
    echo abs(-5);      // 5
    echo pow(2, 3);    // 8
    echo sqrt(16);     // 4
    echo round(3.7);   // 4
?>
