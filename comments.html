PHP Comments

PHP Comments

Comments are used to explain code and are ignored by the PHP engine. 

// This is a single-line comment
# This is also a single-line comment
/*
   This is a multi-line comment
*/

কোড ব্যাখ্যা করতে কমেন্ট ব্যবহার করা হয়, এগুলো PHP দ্বারা এক্সিকিউট হয় না। 

// এটি এক লাইনের কমেন্ট
# এটিও এক লাইনের কমেন্ট
/*
   এটি মাল্টি-লাইন কমেন্ট
*/
