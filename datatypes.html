PHP Data Types

PHP Data Types

PHP is a loosely typed language, which means you don't need to declare the data type of a variable. PHP automatically converts the variable to the correct data type based on its value. However, understanding data types is crucial for writing efficient and bug-free code.

1. Scalar Data Types

Scalar data types can hold only one value at a time:

1.1 String

A string is a sequence of characters enclosed in quotes. Strings can be created using single quotes, double quotes, or heredoc syntax. 

<?php
// Single quotes
$name = 'John Doe';

// Double quotes (allows variable interpolation)
$greeting = "Hello, $name!";

// Heredoc syntax
$long_text = <<<EOT
This is a multi-line string
using heredoc syntax.
EOT;

echo $name . "<br>";
echo $greeting . "<br>";
echo $long_text;
?>

1.2 Integer

An integer is a whole number (positive, negative, or zero) without decimals. PHP supports different number systems. 

<?php
// Decimal numbers
$x = 42;
$y = -17;
$z = 0;

// Hexadecimal numbers (prefixed with 0x)
$hex = 0x2A;  // 42 in decimal

// Octal numbers (prefixed with 0)
$octal = 052; // 42 in decimal

// Binary numbers (prefixed with 0b)
$binary = 0b101010; // 42 in decimal

echo "Decimal: $x<br>";
echo "Hexadecimal: $hex<br>";
echo "Octal: $octal<br>";
echo "Binary: $binary<br>";
?>

1.3 Float (Double)

A float is a number with a decimal point or in exponential form. PHP uses double-precision floating-point format. 

<?php
// Decimal notation
$price = 19.99;
$pi = 3.14159;

// Exponential notation
$large = 2.5e3;    // 2500
$small = 7E-10;    // 0.0000000007

// Scientific notation
$scientific = 1.23e-4; // 0.000123

echo "Price: $price<br>";
echo "Pi: $pi<br>";
echo "Large number: $large<br>";
echo "Small number: $small<br>";
echo "Scientific: $scientific<br>";
?>

1.4 Boolean

A boolean represents two possible states: true or false. Booleans are often used in conditional testing. 

<?php
$is_logged_in = true;
$is_admin = false;
$has_permission = true;

// Boolean in conditional statements
if ($is_logged_in) {
    echo "Welcome back!<br>";
}

if ($is_admin) {
    echo "Admin privileges granted.<br>";
} else {
    echo "Regular user access.<br>";
}

// Boolean operations
$result = $is_logged_in && $has_permission;
echo "Can access: " . ($result ? "Yes" : "No");
?>

2. Compound Data Types

Compound data types can hold multiple values:

2.1 Array

An array is a collection of values stored under a single variable name. PHP supports indexed arrays, associative arrays, and multidimensional arrays. 

<?php
// Indexed array
$fruits = array("apple", "banana", "orange");
$colors = ["red", "green", "blue"]; // Short syntax

// Associative array
$person = array(
    "name" => "John",
    "age" => 30,
    "city" => "New York"
);

// Multidimensional array
$students = array(
    array("name" => "Alice", "grade" => 85),
    array("name" => "Bob", "grade" => 92),
    array("name" => "Charlie", "grade" => 78)
);

echo "First fruit: " . $fruits[0] . "<br>";
echo "Person's name: " . $person["name"] . "<br>";
echo "Alice's grade: " . $students[0]["grade"] . "<br>";
?>

2.2 Object

An object is an instance of a class. Objects contain properties (variables) and methods (functions). 

<?php
class Car {
    // Properties
    public $brand;
    public $model;
    public $year;
    
    // Constructor
    public function __construct($brand, $model, $year) {
        $this->brand = $brand;
        $this->model = $model;
        $this->year = $year;
    }
    
    // Method
    public function getInfo() {
        return "This is a {$this->year} {$this->brand} {$this->model}.";
    }
    
    public function isOld() {
        return (date("Y") - $this->year) > 10;
    }
}

// Creating objects
$car1 = new Car("Toyota", "Camry", 2020);
$car2 = new Car("Honda", "Civic", 2010);

echo $car1->getInfo() . "<br>";
echo $car2->getInfo() . "<br>";
echo "Is car2 old? " . ($car2->isOld() ? "Yes" : "No");
?>

3. Special Data Types

3.1 NULL

NULL is a special data type that represents a variable with no value or an unset variable. 

<?php
// Explicitly setting to NULL
$var1 = NULL;

// Variable declared but not assigned
$var2;

// Unsetting a variable
$var3 = "Hello";
unset($var3);

// Checking for NULL
if (is_null($var1)) {
    echo "var1 is NULL<br>";
}

if (!isset($var2)) {
    echo "var2 is not set<br>";
}

// Using null coalescing operator
$name = $var1 ?? "Default Name";
echo "Name: $name<br>";
?>

3.2 Resource

A resource is a special variable that holds a reference to an external resource, such as a database connection or file handle. 

<?php
// File resource
$file = fopen("example.txt", "r");
if ($file) {
    echo "File opened successfully<br>";
    echo "Resource type: " . gettype($file) . "<br>";
    fclose($file);
}

// echo "Connection type: " . gettype($connection);
?>

4. Type Checking and Conversion

4.1 Type Checking Functions

<?php
$string = "Hello";
$integer = 42;
$float = 3.14;
$boolean = true;
$array = [1, 2, 3];
$null = NULL;

// Type checking functions
echo "is_string(): " . (is_string($string) ? "true" : "false") . "<br>";
echo "is_int(): " . (is_int($integer) ? "true" : "false") . "<br>";
echo "is_float(): " . (is_float($float) ? "true" : "false") . "<br>";
echo "is_bool(): " . (is_bool($boolean) ? "true" : "false") . "<br>";
echo "is_array(): " . (is_array($array) ? "true" : "false") . "<br>";
echo "is_null(): " . (is_null($null) ? "true" : "false") . "<br>";

// gettype() function
echo "String type: " . gettype($string) . "<br>";
echo "Integer type: " . gettype($integer) . "<br>";
?>

4.2 Type Conversion

<?php
// Automatic type conversion
$x = "10";
$y = 5;

echo "String + Integer: " . ($x + $y) . "<br>"; // 15 (integer)

// Explicit type casting
$string_num = "42.5";
$integer_cast = (int)$string_num;
$float_cast = (float)$string_num;
$string_cast = (string)$integer_cast;

echo "Original: $string_num<br>";
echo "As integer: $integer_cast<br>";
echo "As float: $float_cast<br>";
echo "As string: $string_cast<br>";

// Boolean conversion
$truthy = "Hello";
$falsy = "";

echo "Truthy as bool: " . (bool)$truthy . "<br>";
echo "Falsy as bool: " . (bool)$falsy . "<br>";
?>

5. Variable Scope and Lifetime

<?php
// Global variable
$global_var = "I'm global";

function testFunction() {
    global $global_var; // Access global variable
    $local_var = "I'm local";
    
    echo "Global: $global_var<br>";
    echo "Local: $local_var<br>";
}

testFunction();
echo "Outside function: $global_var<br>";
// echo $local_var; // This would cause an error
?>
