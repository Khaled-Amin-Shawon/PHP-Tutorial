PHP Data Types

PHP Variable Types

PHP supports several different data types for variables:

  • String – a sequence of characters, e.g. "Hello"
  • Integer – whole numbers, e.g. 42
  • Float (or double) – numbers with decimals, e.g. 3.14
  • Boolean – true or false
  • Array – a collection of values, e.g. array(1, 2, 3)
  • Object – an instance of a class
  • NULL – a variable with no value
  • Resource – a special variable holding a reference to an external resource (like a database connection) 
<?php
// String
$name = "Alice";

// Integer
$age = 30;

// Float
$price = 19.99;

// Boolean
$is_admin = true;

// Array
$colors = array("red", "green", "blue");

// Object
class Car {
    public $model;
    function __construct($model) {
        $this->model = $model;
    }
}
$myCar = new Car("Toyota");

// NULL
$nothing = NULL;
?>

Get the Type of a Variable

You can use the gettype() function in PHP to find out the type of a variable. 

<?php
$var = "Hello, World!";
echo gettype($var); // Outputs: string
?>

PHP Integer

An integer is a whole number (positive, negative, or zero) without decimals. In PHP, integers can be specified in decimal, hexadecimal, or octal notation. 

<?php
$x = 42;         // decimal number
$y = 0x2A;       // hexadecimal number (42 in decimal)
$z = 052;        // octal number (42 in decimal)

echo $x; // Outputs: 42
echo $y; // Outputs: 42
echo $z; // Outputs: 42
?>

ইন্টিজার হচ্ছে সম্পূর্ণ সংখ্যা (দশমিক ছাড়া)। PHP-তে ইন্টিজার দশমিক, হেক্সাডেসিমাল বা অক্টাল ফরম্যাটে লেখা যায়।

PHP Float

A float (also called double or floating point number) is a number with a decimal point or in exponential form. 

<?php
$a = 3.1416;      // float
$b = 2.5e3;       // float (2500)
$c = 7E-10;       // float (0.0000000007)

echo $a; // Outputs: 3.1416
echo $b; // Outputs: 2500
echo $c; // Outputs: 7.0E-10
?>

ফ্লোট হচ্ছে দশমিক সংখ্যা বা এক্সপোনেনশিয়াল ফরম্যাটে লেখা সংখ্যা। যেমন: 3.14, 2.5e3 ইত্যাদি।

PHP Boolean

A boolean represents two possible states: true or false. Booleans are often used in conditional testing. 

<?php
$is_logged_in = true;
$is_admin = false;

if ($is_logged_in) {
    echo "Welcome!";
} else {
    echo "Please log in.";
}
?>

বুলিয়ান ডেটা টাইপে শুধু দুইটি মান থাকতে পারে: true (সত্য) অথবা false (মিথ্যা)। সাধারণত শর্ত যাচাই করার জন্য ব্যবহার হয়।

PHP Object

In PHP, an object is an instance of a class. Classes are templates that define properties (variables) and methods (functions) that objects created from the class will have. 

<?php
class Car {
    public $model;
    public $color;

    // Constructor function
    function __construct($model, $color) {
        $this->model = $model;
        $this->color = $color;
    }

    function getMessage() {
        return "The car is a " . $this->color . " " . $this->model . ".";
    }
}

$car1 = new Car("Volvo", "red");
$car2 = new Car("BMW", "black");

echo $car1->getMessage(); // Outputs: The car is a red Volvo.
echo "<br>";
echo $car2->getMessage(); // Outputs: The car is a black BMW.
?>

অবজেক্ট হচ্ছে ক্লাসের একটি নির্দিষ্ট উদাহরণ (instance)। ক্লাসে যেসব প্রপার্টি ও মেথড থাকে, অবজেক্ট সেগুলো পায়। উপরের উদাহরণে Car ক্লাসে $model ও $color প্রপার্টি এবং getMessage() মেথড আছে। __construct() ফাংশনটি অবজেক্ট তৈরি হওয়ার সময় স্বয়ংক্রিয়ভাবে কল হয় এবং প্রপার্টিগুলো সেট করে। প্রতিটি অবজেক্টের মান আলাদা হতে পারে।

PHP NULL Value

The NULL data type in PHP can have only one value: NULL. A variable is of type NULL if it has no value assigned to it. 

<?php
$var1 = NULL;
$var2; // No value assigned, so it is NULL by default

echo $var1; // Outputs nothing
var_dump($var1); // Outputs: NULL

var_dump($var2); // Outputs: NULL

// You can empty a variable by setting it to NULL
$var3 = "Hello";
$var3 = NULL;
var_dump($var3); // Outputs: NULL
?>

NULL একটি বিশেষ ডেটা টাইপ, যার একমাত্র মান হচ্ছে NULL। কোনো ভ্যারিয়েবলের কোনো মান না থাকলে সেটি NULL হয়। যদি কোনো ভ্যারিয়েবল ডিক্লেয়ার করে মান না দেওয়া হয়, তাহলে সেটি স্বয়ংক্রিয়ভাবে NULL হয়ে যায়। আবার, কোনো ভ্যারিয়েবলের মান মুছে ফেলতে চাইলে সেটিকে NULL সেট করতে পারেন।

Change Data Type

If you assign an integer value to a variable, the type will automatically be an integer.
If you assign a string to the same variable, the type will change to a string: 

<?php
$x = 10;           // $x is now an integer
var_dump($x);      // Outputs: int(10)

$x = "Hello PHP";  // $x is now a string
var_dump($x);      // Outputs: string(9) "Hello PHP"
?>

যদি কোনো ভ্যারিয়েবলে ইন্টিজার মান অ্যাসাইন করেন, তাহলে সেটি স্বয়ংক্রিয়ভাবে ইন্টিজার টাইপ হয়ে যায়।
আবার, একই ভ্যারিয়েবলে স্ট্রিং মান অ্যাসাইন করলে টাইপ পরিবর্তন হয়ে স্ট্রিং হয়ে যাবে।
