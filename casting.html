PHP Casting

PHP Casting

Type casting in PHP means converting a variable from one data type to another. This is useful when you want to ensure a value is treated as a specific type, such as integer, float, string, etc.

How to Cast Types in PHP?

You can cast a variable to another type by placing the type in parentheses before the variable. 

<?php
$number = "42";
$intNumber = (int)$number; // Cast to integer
$floatNumber = (float)$number; // Cast to float
$stringNumber = (string)$number; // Cast to string

echo $intNumber;    // Output: 42
echo $floatNumber;  // Output: 42
echo $stringNumber; // Output: 42
?>

Available Cast Types

  • (int) or (integer) — Cast to integer
  • (float) or (double) or (real) — Cast to float
  • (string) — Cast to string
  • (bool) or (boolean) — Cast to boolean
  • (array) — Cast to array
  • (object) — Cast to object
  • (unset) — Cast to NULL

Example: Casting String to Integer

<?php
$value = "123abc";
$intValue = (int)$value;
echo $intValue; // Output: 123
?>

Example: Casting Float to Integer

<?php
$floatValue = 5.99;
$intValue = (int)$floatValue;
echo $intValue; // Output: 5
?>

Example: Casting to Boolean

<?php
$val1 = 0;
$val2 = 42;
$val3 = "";

echo (bool)$val1; // Output: (nothing, false)
echo (bool)$val2; // Output: 1 (true)
echo (bool)$val3; // Output: (nothing, false)
?>

Type Casting Table

Cast Type Syntax Result Example Description
Integer (int)$var (int)"12.5" → 12 Converts to integer (removes decimal, parses from string start)
Float (float)$var (float)"12.5" → 12.5 Converts to floating point number
String (string)$var (string)123 → "123" Converts to string
Boolean (bool)$var (bool)0 → false Converts to true/false
Array (array)$var (array)5 → [5] Converts to array
Object (object)$var (object)5 → object with property 'scalar' Converts to object
NULL (unset)$var (unset)5 → NULL Converts to NULL (rarely used)

Automatic Type Juggling

PHP automatically converts types when needed (called "type juggling"). For example, adding a string and a number: 

<?php
$a = "10";
$b = 5;
$c = $a + $b; // $a is automatically cast to integer
echo $c; // Output: 15
?>

However, for strict type conversion, use explicit casting as shown above.

বাংলা ব্যাখ্যা (Bangla Explanation)

PHP-তে কাস্টিং মানে হচ্ছে একটি ভেরিয়েবলকে এক ডেটা টাইপ থেকে অন্য ডেটা টাইপে রূপান্তর করা। এটি তখন দরকার হয় যখন আপনি চান কোনো মান নির্দিষ্ট টাইপে ব্যবহার করতে। যেমন, স্ট্রিংকে ইন্টিজার, ফ্লোট, বুলিয়ান ইত্যাদিতে কাস্ট করা যায়।

PHP-তে কাস্টিং করার নিয়ম
  • (int) বা (integer) — ইন্টিজার টাইপে কাস্ট করে। যেমন: (int)"12.5" আউটপুট: 12
  • (float) — ফ্লোট টাইপে কাস্ট করে। যেমন: (float)"12.5" আউটপুট: 12.5
  • (string) — স্ট্রিং টাইপে কাস্ট করে। যেমন: (string)123 আউটপুট: "123"
  • (bool) — বুলিয়ান টাইপে কাস্ট করে। যেমন: (bool)0 আউটপুট: false
  • (array) — অ্যারে টাইপে কাস্ট করে। যেমন: (array)5 আউটপুট: [5]
  • (object) — অবজেক্ট টাইপে কাস্ট করে। যেমন: (object)5 আউটপুট: একটি অবজেক্ট
  • (unset) — NULL-এ কাস্ট করে (খুব কম ব্যবহৃত হয়)
কাস্টিং-এর কিছু উদাহরণ
<?php
// স্ট্রিং থেকে ইন্টিজার
$text = "100.5";
$number = (int)$text; // 100

// ফ্লোট থেকে ইন্টিজার
$floatVal = 7.99;
$intVal = (int)$floatVal; // 7

// ইন্টিজার থেকে বুলিয়ান
$flag = (bool)$number; // true (1)

// অ্যারে কাস্টিং
$val = 5;
$arr = (array)$val; // [5]

// অবজেক্ট কাস্টিং
$obj = (object)$val; // stdClass Object ( [scalar] => 5 )
?>
কিছু গুরুত্বপূর্ণ বিষয়
  • কাস্টিং করার জন্য টাইপটি ব্র্যাকেটের মধ্যে লিখে ভেরিয়েবলের আগে বসাতে হয়।
  • স্ট্রিং থেকে ইন্টিজার কাস্ট করলে, স্ট্রিং-এর শুরুতে যতক্ষণ সংখ্যা থাকবে, ততক্ষণ পর্যন্ত সংখ্যা নেয়।
  • বুলিয়ান কাস্টিং: 0, "0", "", NULL, [], ইত্যাদি false; অন্য সব true।
  • PHP স্বয়ংক্রিয়ভাবে অনেক সময় টাইপ পরিবর্তন করে (type juggling), তবে নির্দিষ্ট টাইপ চাইলে কাস্টিং ব্যবহার করুন।
