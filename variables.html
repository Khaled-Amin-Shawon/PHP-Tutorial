Learn about variables in PHP programming

PHP Variables

In PHP, a variable is used to store information that can be used and changed throughout your script. Variable names can be short (like $x, $y) or more descriptive (like $age, $carname, $total_volume).

Rules for PHP Variables

  • A variable starts with the $ sign, followed by the name of the variable
  • A variable name must start with a letter or the underscore character (_)
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _)
  • Variable names are case-sensitive ($age and $AGE are different variables)

Example:

<?php
$txt = "Hello, World!";
$x = 5;
$y = 10.5;

echo $txt; // Outputs: Hello, World!
echo $x;   // Outputs: 5
echo $y;   // Outputs: 10.5
?>

Valid and Invalid Variable Names

  • $name ✔️ valid
  • $_name ✔️ valid
  • $name1 ✔️ valid
  • $1name ❌ invalid
  • $na-me ❌ invalid
  • $na me ❌ invalid

PHP-তে ভ্যারিয়েবল ডিক্লেয়ার করতে $ চিহ্ন দিয়ে শুরু করতে হয়। ভ্যারিয়েবল নাম অবশ্যই একটি অক্ষর বা আন্ডারস্কোর (_) দিয়ে শুরু হবে, সংখ্যা দিয়ে শুরু করা যাবে না। নামের মধ্যে শুধু অক্ষর, সংখ্যা ও আন্ডারস্কোর থাকতে পারে। ভ্যারিয়েবল নাম কেস-সেন্সিটিভ, অর্থাৎ $age এবং $AGE আলাদা ভ্যারিয়েবল।

Output Variables

You can output the value of a variable in PHP using the echo or print statement. 

<?php
$txt = "PHP is awesome!";
echo $txt; // Outputs: PHP is awesome!
?>

PHP is a Loosely Typed Language

In the example above, notice that we did not have to tell PHP which data type the variable is.

PHP automatically associates a data type to the variable, depending on its value. Since the data types are not set in a strict sense, you can do things like adding a string to an integer without causing an error. 

<?php
$x = 5;         // integer
$y = "10 cats"; // string

$z = $x + $y;   // $y is converted to integer (10), so $z = 15

echo $z; // Outputs: 15
?>

PHP একটি loosely typed ভাষা, অর্থাৎ ভ্যারিয়েবল ডিক্লেয়ার করার সময় ডেটা টাইপ আলাদাভাবে জানাতে হয় না। PHP নিজে থেকেই ভ্যালু দেখে টাইপ নির্ধারণ করে। তাই আপনি চাইলে স্ট্রিং ও নাম্বার একসাথে যোগ করতে পারেন, PHP স্বয়ংক্রিয়ভাবে কনভার্ট করে নেয়।

Assign String to a Variable

To assign a string value to a variable in PHP, use the = operator and wrap the string in quotes. 

<?php
$message = "Welcome to PHP!";
echo $message; // Outputs: Welcome to PHP!
?>

Assign Multiple Values to Variables

You can assign multiple values to multiple variables in one line using the list() function or by separate assignments. 

<?php
// Assigning multiple values using list()
list($a, $b, $c) = array("Apple", "Banana", "Cherry");
echo $a; // Outputs: Apple
echo $b; // Outputs: Banana
echo $c; // Outputs: Cherry

// Or assign separately
$x = "Red";
$y = "Green";
$z = "Blue";
?>

PHP Variable Scope

In PHP, the scope of a variable determines where it can be accessed in your script. There are three main types of variable scope:

  • Local - Variables declared inside a function and accessible only within that function.
  • Global - Variables declared outside any function and accessible everywhere except inside functions (unless specified).
  • Static - Variables within a function that retain their value between function calls.

Local Scope

A variable declared inside a function has a local scope and can only be accessed within that function. 

<?php
function myTest() {
    $x = 5; // local scope
    echo $x; // Outputs: 5
}
myTest();
// echo $x; // Error: $x is not accessible outside the function
?>

Global Scope

A variable declared outside a function has a global scope and can be accessed anywhere outside functions. To use a global variable inside a function, use the global keyword. 

<?php
$x = 10; // global scope

function myTest() {
    global $x;
    echo $x; // Outputs: 10
}
myTest();
?>

Static Scope

Normally, when a function is completed, all its variables are deleted. But if you declare a variable as static, it will retain its value between function calls. 

<?php
function myCounter() {
    static $count = 0;
    $count++;
    echo $count . " ";
}
myCounter(); // Outputs: 1
myCounter(); // Outputs: 2
myCounter(); // Outputs: 3
?>

PHP-তে ভ্যারিয়েবলের স্কোপ নির্ধারণ করে কোথায় সেই ভ্যারিয়েবলটি ব্যবহার করা যাবে। তিন ধরনের স্কোপ আছে: local (ফাংশনের ভিতরে), global (ফাংশনের বাইরে), এবং static (ফাংশনের ভিতরে, কিন্তু মান ধরে রাখে)। 

<?php
// লোকাল স্কোপ
function myTest() {
    $x = ৫;
    echo $x; // আউটপুট: ৫
}
myTest();

// গ্লোবাল স্কোপ
$x = ১০;
function myTest2() {
    global $x;
    echo $x; // আউটপুট: ১০
}
myTest2();

// স্ট্যাটিক স্কোপ
function myCounter() {
    static $count = ০;
    $count++;
    echo $count . " ";
}
myCounter(); // আউটপুট: ১
myCounter(); // আউটপুট: ২
myCounter(); // আউটপুট: ৩
?>
