PHP Variables

Variables in PHP start with a $ sign, followed by the variable name. They are used to store data. 

<?php
    $txt = "Hello World!";
    $x = 5;
    $y = 10.5;
    echo $txt;
?>

PHP-তে ভেরিয়েবল $ চিহ্ন দিয়ে শুরু হয় এবং ডেটা সংরক্ষণে ব্যবহৃত হয়। 

<?php
    $txt = "হ্যালো ওয়ার্ল্ড!";
    $x = ৫;
    $y = ১০.৫;
    echo $txt;
?>
