PHP Variables

In PHP, a variable is used to store information that can be used and changed throughout your script. Variable names can be short (like $x, $y) or more descriptive (like $age, $carname, $total_volume).

Rules for PHP Variables

  • A variable starts with the $ sign, followed by the name of the variable
  • A variable name must start with a letter or the underscore character (_)
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _)
  • Variable names are case-sensitive ($age and $AGE are different variables)

Example:

<?php
$txt = "Hello, World!";
$x = 5;
$y = 10.5;

echo $txt; // Outputs: Hello, World!
echo $x;   // Outputs: 5
echo $y;   // Outputs: 10.5
?>

Valid and Invalid Variable Names

  • $name ✔️ valid
  • $_name ✔️ valid
  • $name1 ✔️ valid
  • $1name ❌ invalid
  • $na-me ❌ invalid
  • $na me ❌ invalid

PHP-তে ভ্যারিয়েবল ডিক্লেয়ার করতে $ চিহ্ন দিয়ে শুরু করতে হয়। ভ্যারিয়েবল নাম অবশ্যই একটি অক্ষর বা আন্ডারস্কোর (_) দিয়ে শুরু হবে, সংখ্যা দিয়ে শুরু করা যাবে না। নামের মধ্যে শুধু অক্ষর, সংখ্যা ও আন্ডারস্কোর থাকতে পারে। ভ্যারিয়েবল নাম কেস-সেন্সিটিভ, অর্থাৎ $age এবং $AGE আলাদা ভ্যারিয়েবল।

Output Variables

You can output the value of a variable in PHP using the echo or print statement. 

<?php
$txt = "PHP is awesome!";
echo $txt; // Outputs: PHP is awesome!
?>

PHP is a Loosely Typed Language

In the example above, notice that we did not have to tell PHP which data type the variable is.

PHP automatically associates a data type to the variable, depending on its value. Since the data types are not set in a strict sense, you can do things like adding a string to an integer without causing an error. 

<?php
$x = 5;         // integer
$y = "10 cats"; // string

$z = $x + $y;   // $y is converted to integer (10), so $z = 15

echo $z; // Outputs: 15
?>

PHP একটি loosely typed ভাষা, অর্থাৎ ভ্যারিয়েবল ডিক্লেয়ার করার সময় ডেটা টাইপ আলাদাভাবে জানাতে হয় না। PHP নিজে থেকেই ভ্যালু দেখে টাইপ নির্ধারণ করে। তাই আপনি চাইলে স্ট্রিং ও নাম্বার একসাথে যোগ করতে পারেন, PHP স্বয়ংক্রিয়ভাবে কনভার্ট করে নেয়।
