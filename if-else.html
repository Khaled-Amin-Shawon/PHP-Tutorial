PHP If...Else...Elseif

PHP Conditional Statements

Very often when you write code, you want to perform different actions for different conditions. You can use conditional statements in your code to do this.

প্রায়ই কোড লেখার সময়, বিভিন্ন শর্ত অনুযায়ী বিভিন্ন কাজ করতে হয়। PHP-তে কন্ডিশনাল স্টেটমেন্ট ব্যবহার করে আপনি এটি করতে পারেন।

Types of Conditional Statements in PHP

  • if statement – executes some code if one condition is true
    একটি শর্ত সত্য হলে কিছু কোড চালায়।
  • if...else statement – executes some code if a condition is true and another code if that condition is false
    শর্ত সত্য হলে একটি কোড, মিথ্যা হলে অন্য কোড চালায়।
  • if...elseif...else statement – executes different codes for more than two conditions
    দুইয়ের বেশি শর্তের জন্য আলাদা কোড চালাতে ব্যবহৃত হয়।
  • switch statement – selects one of many blocks of code to be executed
    একাধিক কোড ব্লকের মধ্যে একটি নির্বাচন করে চালায়।

PHP if Statement

The if statement executes some code if a condition is true. 

<?php
$t = 20;
if ($t < 30) {
    echo "Have a nice day!";
}
?>

if স্টেটমেন্ট একটি শর্ত সত্য হলে কোড চালায়।
উপরের উদাহরণে, $t যদি ৩০-এর কম হয়, তাহলে "Have a nice day!" প্রিন্ট হবে।

PHP if...else Statement

The if...else statement executes one block of code if a condition is true, and another block if it is false. 

<?php
$age = 18;
if ($age >= 18) {
    echo "You are an adult.";
} else {
    echo "You are a minor.";
}
?>

if...else স্টেটমেন্টে, শর্ত সত্য হলে একটি কোড, মিথ্যা হলে অন্য কোড চালায়।
এখানে, $age ১৮ বা তার বেশি হলে "You are an adult." দেখাবে, না হলে "You are a minor."।

PHP if...elseif...else Statement

The if...elseif...else statement allows you to test several conditions. 

<?php
$score = 75;
if ($score >= 90) {
    echo "Grade: A";
} elseif ($score >= 80) {
    echo "Grade: B";
} elseif ($score >= 70) {
    echo "Grade: C";
} else {
    echo "Grade: F";
}
?>

if...elseif...else দিয়ে একাধিক শর্ত পরীক্ষা করা যায়।
এখানে, $score অনুযায়ী গ্রেড নির্ধারণ করা হয়েছে।

PHP Shorthand if Statements (Ternary Operator)

PHP provides a shorthand way to write simple if...else statements using the ternary operator ?:. 

<?php
$age = 16;
$message = ($age >= 18) ? "You are an adult." : "You are a minor.";
echo $message; // Outputs: You are a minor.
?>

PHP-তে if...else স্টেটমেন্ট সংক্ষেপে লেখার জন্য ternary operator ?: ব্যবহার করা হয়।
উপরের উদাহরণে, $age ১৮ বা তার বেশি হলে $message হবে "You are an adult.", না হলে "You are a minor."।

Short Ternary (Null Coalescing) Operator

PHP 7.0+ introduces the null coalescing operator ?? to provide a default value if a variable is not set or is NULL. 

<?php
$username = $_GET['user'] ?? "Guest";
echo $username;
?>

PHP 7.0 থেকে ?? (null coalescing operator) ব্যবহার করে সহজেই ডিফল্ট মান সেট করা যায়।
যেমন, $_GET['user'] সেট না থাকলে $username হবে "Guest"।
