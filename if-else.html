PHP If...Else...Elseif

Conditional statements are used to perform actions based on conditions. Use if, else, and elseif to control the flow.

<?php
    $x = 10;
    if ($x > 5) {
        echo "x is greater than 5";
    } elseif ($x == 5) {
        echo "x is 5";
    } else {
        echo "x is less than 5";
    }
?>
