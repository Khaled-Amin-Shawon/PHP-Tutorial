PHP Loops

Loops are used to execute the same block of code multiple times. Common loops in PHP are for, while, and foreach.

for loop

<?php
    for ($i = 1; $i <= 3; $i++) {
        echo $i;
    }
?>
123

while loop

<?php
    $i = 1;
    while ($i <= 3) {
        echo $i;
        $i++;
    }
?>
123

foreach loop

<?php
    $arr = ["a", "b", "c"];
    foreach ($arr as $val) {
        echo $val;
    }
?>
abc
