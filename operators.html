PHP Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, etc.

Operator Name Example
+ Addition $a + $b
- Subtraction $a - $b
* Multiplication $a * $b
/ Division $a / $b
% Modulus $a % $b 
<?php
$a = 10;
$b = 3;
echo $a + $b; // 13
echo $a % $b; // 1
?>

Assignment Operators

Assignment operators are used to assign values to variables.

Operator Name Example
= Assign $a = 5
+= Add and assign $a += 2
-= Subtract and assign $a -= 2
*= Multiply and assign $a *= 2
/= Divide and assign $a /= 2 
<?php
$a = 5;
$a += 3; // $a is now 8
?>

Comparison Operators

Comparison operators are used to compare two values and return true or false.

Operator Name Example
== Equal $a == $b
!= Not equal $a != $b
< Less than $a < $b
> Greater than $a > $b
<= Less than or equal $a <= $b
>= Greater than or equal $a >= $b 
<?php
$a = 5;
$b = 8;
if ($a < $b) {
    echo "a is less than b";
}
?>

Logical Operators

Logical operators are used to combine conditional statements.

Operator Name Example
&& And $a && $b
|| Or $a || $b
! Not !$a 
<?php
$a = true;
$b = false;
if ($a && !$b) {
    echo "a is true and b is false";
}
?>
Note: Operators are essential in PHP for performing calculations, making comparisons, and controlling logic. Mastering operators is key to writing effective PHP code.
