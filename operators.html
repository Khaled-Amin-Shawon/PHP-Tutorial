PHP Operators

PHP Operators

Operators are used to perform operations on variables and values in PHP.

PHP-তে অপারেটর ব্যবহার করে ভ্যারিয়েবল ও মানের উপর বিভিন্ন ধরনের অপারেশন করা যায়।

Types of Operators in PHP

  • Arithmetic Operators – Used to perform common arithmetic operations.
    গাণিতিক অপারেশন করার জন্য ব্যবহৃত হয়।
  • Assignment Operators – Used to assign values to variables.
    ভ্যারিয়েবলে মান অ্যাসাইন করতে ব্যবহৃত হয়।
  • Comparison Operators – Used to compare two values.
    দুইটি মান তুলনা করতে ব্যবহৃত হয়।
  • Increment/Decrement Operators – Used to increment or decrement a variable's value.
    ভ্যারিয়েবলের মান বাড়াতে বা কমাতে ব্যবহৃত হয়।
  • Logical Operators – Used to combine conditional statements.
    একাধিক শর্ত একত্রে যাচাই করতে ব্যবহৃত হয়।
  • String Operators – Used to combine or manipulate strings.
    স্ট্রিং একত্রে যোগ বা পরিবর্তন করতে ব্যবহৃত হয়।
  • Array Operators – Used to compare arrays.
    দুইটি অ্যারে তুলনা করতে ব্যবহৃত হয়।
  • Conditional Assignment Operators – Used to set a value depending on conditions.
    শর্ত অনুযায়ী মান নির্ধারণ করতে ব্যবহৃত হয়।
Note: Each operator group has its own syntax and use cases. The following sections will explain each group in detail with examples.
নোট: প্রতিটি অপারেটরের নিজস্ব ব্যবহার ও সিনট্যাক্স আছে। নিচে প্রতিটি অপারেটর গ্রুপের বিস্তারিত উদাহরণ দেয়া হয়েছে।

1. Arithmetic Operators

Arithmetic operators are used to perform common mathematical operations.

গাণিতিক অপারেটর দিয়ে যোগ, বিয়োগ, গুণ, ভাগ ইত্যাদি করা যায়।

Operator Name Example Result
+ Addition
যোগ		 $a + $b Sum of $a and $b
- Subtraction
বিয়োগ		 $a - $b Difference of $a and $b
* Multiplication
গুণ		 $a * $b Product of $a and $b
/ Division
ভাগ		 $a / $b Quotient of $a and $b
% Modulus
ভাগশেষ		 $a % $b Remainder of $a divided by $b
** Exponentiation
ঘাত		 $a ** $b $a to the power of $b 
<?php
$a = 10;
$b = 3;
echo $a + $b; // 13
echo $a - $b; // 7
echo $a * $b; // 30
echo $a / $b; // 3.333...
echo $a % $b; // 1
echo $a ** $b; // 1000
?>

উপরের কোডে প্রতিটি গাণিতিক অপারেটরের ব্যবহার দেখানো হয়েছে।

2. Assignment Operators

Assignment operators are used to assign values to variables.

অ্যাসাইনমেন্ট অপারেটর দিয়ে ভ্যারিয়েবলে মান সেট করা হয়।

Operator Example Same As Description
= $x = 5 Assigns 5 to $x
+= $x += 3 $x = $x + 3 Adds 3 to $x
-= $x -= 2 $x = $x - 2 Subtracts 2 from $x
*= $x *= 4 $x = $x * 4 Multiplies $x by 4
/= $x /= 5 $x = $x / 5 Divides $x by 5
%= $x %= 3 $x = $x % 3 Assigns remainder of $x divided by 3 
<?php
$x = 10;
$x += 5; // $x = 15
$x -= 3; // $x = 12
$x *= 2; // $x = 24
$x /= 4; // $x = 6
$x %= 4; // $x = 2
?>

এখানে প্রতিটি অ্যাসাইনমেন্ট অপারেটরের ব্যবহার দেখানো হয়েছে।

3. Comparison Operators

Comparison operators are used to compare two values.

তুলনা করার জন্য কম্পারিজন অপারেটর ব্যবহার করা হয়।

Operator Name Example Result
== Equal $a == $b True if $a is equal to $b
=== Identical $a === $b True if $a is equal to $b and of same type
!= Not equal $a != $b True if $a is not equal to $b
<> Not equal $a <> $b True if $a is not equal to $b
!== Not identical $a !== $b True if $a is not equal or not same type as $b
> Greater than $a > $b True if $a is greater than $b
< Less than $a < $b True if $a is less than $b
>= Greater than or equal to $a >= $b True if $a is greater than or equal to $b
<= Less than or equal to $a <= $b True if $a is less than or equal to $b
<=> Spaceship (PHP 7+) $a <=> $b -1, 0, or 1 
<?php
$a = 5;
$b = "5";
$c = 10;

var_dump($a == $b);   // true (value equal)
var_dump($a === $b);  // false (type not equal)
var_dump($a != $c);   // true
var_dump($a < $c);    // true
var_dump($a >= $b);   // true
var_dump($a <=> $c); // -1 ($a < $c)
?>

এখানে বিভিন্ন কম্পারিজন অপারেটরের ব্যবহার দেখানো হয়েছে।

4. Increment/Decrement Operators

These operators are used to increment or decrement a variable's value.

ভ্যারিয়েবলের মান এক বাড়াতে বা কমাতে ইনক্রিমেন্ট/ডিক্রিমেন্ট অপারেটর ব্যবহার হয়।

Operator Name Example Description
++$x Pre-increment ++$x Increments $x by one, then returns $x
$x++ Post-increment $x++ Returns $x, then increments $x by one
--$x Pre-decrement --$x Decrements $x by one, then returns $x
$x-- Post-decrement $x-- Returns $x, then decrements $x by one 
<?php
$x = 5;
echo ++$x; // 6 (আগে বাড়ায়, পরে দেখায়)
echo $x++; // 6 (আগে দেখায়, পরে বাড়ায়)
echo --$x; // 6 (আগে কমায়, পরে দেখায়)
echo $x--; // 6 (আগে দেখায়, পরে কমায়)
?>

এখানে ইনক্রিমেন্ট ও ডিক্রিমেন্ট অপারেটরের ব্যবহার দেখানো হয়েছে।

5. Logical Operators

Logical operators are used to combine conditional statements.

একাধিক শর্ত একত্রে যাচাই করতে লজিক্যাল অপারেটর ব্যবহার হয়।

Operator Name Example Result
and / && And $a and $b True if both $a and $b are true
or / || Or $a or $b True if either $a or $b is true
xor Xor $a xor $b True if either $a or $b is true, but not both
! Not !$a True if $a is not true 
<?php
$a = true;
$b = false;

var_dump($a && $b); // false
var_dump($a || $b); // true
var_dump($a xor $b); // true
var_dump(!$a); // false
?>

এখানে লজিক্যাল অপারেটরের ব্যবহার দেখানো হয়েছে।

6. String Operators

String operators are used to combine or manipulate strings.

স্ট্রিং অপারেটর দিয়ে স্ট্রিং একত্রে যোগ বা পরিবর্তন করা যায়।

Operator Name Example Description
. Concatenation $a . $b Concatenates $a and $b
.= Concatenation assignment $a .= $b Appends $b to $a 
<?php
$a = "Hello";
$b = " World!";
echo $a . $b; // Hello World!

$a .= $b;
echo $a; // Hello World!
?>

এখানে স্ট্রিং অপারেটরের ব্যবহার দেখানো হয়েছে।

7. Array Operators

Array operators are used to compare arrays.

অ্যারে অপারেটর দিয়ে দুইটি অ্যারে তুলনা করা যায়।

Operator Name Example Description
+ Union $a + $b Union of $a and $b
== Equality $a == $b True if $a and $b have same key/value pairs
=== Identity $a === $b True if $a and $b have same key/value pairs in same order and type
!= Inequality $a != $b True if $a is not equal to $b
<> Inequality $a <> $b True if $a is not equal to $b
!== Non-identity $a !== $b True if $a is not identical to $b 
<?php
$a = array("a" => "red", "b" => "green");
$b = array("c" => "blue", "d" => "yellow");
$c = $a + $b; // Union of $a and $b
var_dump($c);

$d = array("a" => "red", "b" => "green");
var_dump($a == $d);  // true
var_dump($a === $d); // true
?>

এখানে অ্যারে অপারেটরের ব্যবহার দেখানো হয়েছে।

8. Conditional Assignment Operators

Conditional assignment operators are used to set a value depending on conditions.

শর্ত অনুযায়ী মান নির্ধারণ করতে কন্ডিশনাল অ্যাসাইনমেন্ট অপারেটর ব্যবহার হয়।

Operator Name Example Description
?: Ternary $x = ($a > $b) ? $a : $b; If $a > $b, $x = $a, else $x = $b
?? Null coalescing (PHP 7+) $x = $a ?? $b; If $a exists and not null, $x = $a, else $x = $b 
<?php
$a = null;
$b = 10;
$x = ($a !== null) ? $a : $b; // $x = 10

// PHP 7+ null coalescing
$y = $a ?? $b; // $y = 10
?>

এখানে কন্ডিশনাল অ্যাসাইনমেন্ট অপারেটরের ব্যবহার দেখানো হয়েছে।
