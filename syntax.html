PHP Syntax

PHP Syntax

All PHP code is written inside <?php ... ?> tags. Statements end with a semicolon ;. 

<?php
    // PHP code goes here
    echo "Hello, World!";
?>

PHP Case Sensitivity

In PHP, variable names are case-sensitive, but function names and keywords are not case-sensitive.

  • Variables: $name and $NAME are different variables.
  • Functions & Keywords: echo, ECHO, and EcHo all work the same. 
<?php
$color = "red";
$Color = "blue";
echo $color; // Outputs: red
echo $Color; // Outputs: blue

ECHO "This is valid!"; // Outputs: This is valid!
echo "This is also valid!"; // Outputs: This is also valid!
?>

PHP-তে ভ্যারিয়েবল (যেমন $name) কেস-সেন্সিটিভ, অর্থাৎ $name এবং $NAME আলাদা ভ্যারিয়েবল। কিন্তু ফাংশন ও কিওয়ার্ড (যেমন echo) কেস-ইনসেন্সিটিভ, অর্থাৎ echo, ECHO বা EcHo একই কাজ করবে।
