PHP Syntax

All PHP code is written inside <?php ... ?> tags. Statements end with a semicolon ;. 

<?php
    // PHP code goes here
    echo "Hello, World!";
?>

সব PHP কোড <?php ... ?> ট্যাগের মধ্যে লিখতে হয় এবং প্রতিটি স্টেটমেন্ট শেষে ; দিতে হয়।
